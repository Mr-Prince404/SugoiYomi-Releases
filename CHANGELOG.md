# Changelog

Todas las actualizaciones importantes de SugoiYomi se documentarán aquí.

## [1.0.0] - 2026-07-03

### Lanzamiento inicial

- Biblioteca local y remota.
- Lector en modo Página y Cascada.
- Progreso e historial persistentes.
- Descargas CBZ para lectura offline.
- Fuentes instalables y actualizables.
- Búsqueda global.
- Exportación, importación y restauración de datos.
- Recuperación de portadas.
- Instalador optimizado para Windows.
